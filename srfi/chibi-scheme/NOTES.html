Chibi implementation

There are two implementations of this SRFI, Scsh version 0.7, which can be found at GitHub in the scsh repository of scheme, and a Chibi Scheme (srfi 170) library. You can find the Chibi Scheme example implementation, and build notes for scsh, in their own srfi subdirectories.

This file documents the current exceptions and deviations of the Chibi Scheme implementation.

3.1  Error handling

  • The Chibi Scheme implementation's setting and getting of errno are done at the Scheme level, and arbitrary Chibi Scheme code that might change errno can run between SRFI 170 setting, setting by a system or library call due to an error, and SRFI 170 getting the current errno.

3.2  I/O

  • The Chibi Scheme versions of the fdes->*port procedures do not support the buffer-mode argument.

3.3  File system

  • The Chibi Scheme implementation of call-with-temporary-filename is neither finished or exported.
